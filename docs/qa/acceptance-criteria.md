# Acceptance Criteria

## MVP

- [ ] 新規セッションを作成できる
- [ ] 犬/猫を選択できる
- [ ] 名前候補を読み込める
- [ ] 候補名を再生できる
- [ ] 反応あり/弱い/なしを記録できる
- [ ] ランキングを表示できる
- [ ] セッションを保存できる
- [ ] オフラインでも記録できる
- [ ] 権限拒否時に代替説明を表示できる
- [ ] UI文言が「確定」を避けている

## Test Scenarios

- 初回起動
- カメラ権限拒否
- マイク権限拒否
- 通信なし
- 同期失敗
- セッション中断
- 候補辞書空状態
- 大きな文字サイズ
- VoiceOver操作
