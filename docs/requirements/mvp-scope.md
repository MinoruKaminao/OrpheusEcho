# MVP Scope

## MVP Goal

AI自動判定に依存せず、現場で使える「候補名再生・手動反応記録・ランキング保存」機能を最短で提供する。

## MVP Flow

```text
新規セッション
↓
犬/猫選択
↓
個体メモ入力
↓
名前候補リスト読込
↓
候補名を音声再生
↓
反応あり / 弱い / なしを手動記録
↓
スコア順ランキング表示
↓
セッション保存
```

## MVP Features

- 新規セッション作成
- 犬/猫選択
- 任意の個体情報入力
- 名前候補辞書の読込
- TTSまたは収録音声による候補名再生
- 手動反応ボタン
- 試行ログ保存
- ランキング表示
- 履歴表示
- オフライン保存の基本構造

## Deferred Features

- 自動映像解析
- 鳴き声解析
- 学習データ収集モード
- バッチ学習
- モデル配布
- 国際化辞書
- ジョーク機能
