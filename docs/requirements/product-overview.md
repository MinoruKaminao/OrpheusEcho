# Product Overview

## Purpose

保護した犬・猫、または飼い主不明の個体に対して、一般的によく使われる名前候補を音声再生し、対象動物の反応を記録・解析することで、元の呼称に近い名称候補の絞り込みを支援する。

## Positioning

本システムは補助ツールであり、名前の確定、飼い主照合、個体識別、医療診断を行わない。

## Primary Users

- 一般保護者
- 動物保護団体スタッフ
- 自治体施設職員
- 動物病院スタッフ
- 地域見守り担当者
- 学習データ収集に協力する飼い主

## Core Value

- 多数の名前候補を体系的に試せる
- 呼びかけ結果を記録できる
- 反応の強かった候補をランキング化できる
- 複数人で結果を確認・共有できる
- 教師データを蓄積し、将来的にAIモデルを改善できる

## Out of Scope

- 名前の確定保証
- マイクロチップ照会
- 飼い主DBとの自動照合
- 医療診断
- 犬種・猫種の厳密判定
- 人間画像からの実名推定
- 人間画像からの性別・国籍・民族性推定
