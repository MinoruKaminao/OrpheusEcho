# Privacy and Consent

## Sensitive Data

- 動画
- 音声
- 位置情報
- 飼い主情報
- 周辺人物の映り込み
- 学習利用データ

## Consent Rules

- 録画保存は明示同意が必要。
- 音声保存は明示同意が必要。
- 位置情報はデフォルトOFF。
- 学習利用は探索利用とは別同意にする。
- 同意撤回時は学習対象から除外できるようにする。

## Sharing Rules

共有前に以下を確認する。

- 位置情報を含めるか
- 動画・音声を含めるか
- 個人メモを含めるか
- 周辺人物が映っていないか

## Security Requirements

- API認証
- TLS
- 保存データ暗号化
- オブジェクトストレージの署名付きURL
- 監査ログ
- 最小権限
- 削除・保持期間ポリシー
